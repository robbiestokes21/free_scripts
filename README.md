## Versioning

All scripts use `MAJOR.MINOR.PATCH.BUILD[-prerelease]` (e.g. `1.0.0.0`,
`1.2.0-beta`). The published "latest" versions are listed in
[`versions.json`](versions.json).

These are tracked by the `cr_version` update checker as the **free_scripts**
source. To register a new free script there:

1. Add `version '1.0.0.0'` to its `fxmanifest.lua`.
2. Add it to this repo's `versions.json`.
3. Add its name to the `free_scripts` source's `resources` list in
   `cr_version/config.lua`.
